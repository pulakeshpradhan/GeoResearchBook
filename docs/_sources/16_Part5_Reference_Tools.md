# Chapter 5.2: Reference Tools (Zotero)

Stop typing bibliographies manually. Use Zotero.

## Why Zotero?

* It is Free.
* It is Open Source.
* It saves PDFs automatically.

## Crash Course

1. **Install**: Download Zotero and the "Zotero Coordinator" Chrome Extension.
2. **Capture**: When on a Journal Page (e.g., ScienceDirect), click the Zotero button in Chrome. It saves the Title, Author, Year, and PDF.
3. **Cite**: Open Word. Install the Word Plugin.
4. **Write**: "Floods are dangerous." -> Click "Add Citation" -> Search paper -> Enter.
    * *Result*: "Floods are dangerous (Smith, 2020)."
5. **Bibliography**: At the end, click "Add Bibliography".
    * *Result*: It generates the full list A-Z instantly.

## Mendeley

Similar to Zotero. Owned by Elsevier. Good for managing PDFs, but Zotero is better for citations.
